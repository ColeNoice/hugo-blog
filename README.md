# My Hugo Blog

A blog powered by Hugo and synced from Obsidian

## 关于

这个博客使用Hugo构建，内容通过Obsidian进行管理，并自动同步到此仓库。

## 本地开发

```bash
# 克隆仓库
git clone --recursive https://github.com/ColeNoice/hugo-blog.git

# 进入目录
cd hugo-blog

# 启动Hugo服务器
hugo server -D
```

## 部署

此仓库可以直接部署到：
- Vercel
- Netlify
- GitHub Pages
- Cloudflare Pages
